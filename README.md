# Step Counter (Spring Boot)

Simple Spring Boot web app (REST API + static frontend) to track steps in memory.

Build and run:

```bash
mvn spring-boot:run
```

Or build jar:

```bash
mvn package
java -jar target/step-counter-0.0.1-SNAPSHOT.jar
```

Open http://localhost:8080 in a browser.
